# dunst-nctui

Notification center for dunst in the terminal

![](https://i.imgur.com/iAMi2EH.jpg)

## Usage

Copy `config.toml` to `~/.config/dunst-nctui/config.toml`

```
dunst-nctui
```

## Development

```
go run .
```

## Build

```
go build
```
